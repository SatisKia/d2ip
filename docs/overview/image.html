イメージリソースの利用

イメージリソースをアプリケーションに組み込むには、「Finder」からイメージファイルをプロジェクトにドラッグ＆ドロップするだけです。

組み込まれたイメージリソースを使用する

組み込まれたイメージリソースは、

・_Image クラスの load:

メソッドで使用します。例えば次のようになります。

_Image* img_s = [[_Image alloc] init];
[img_s load:@"sample.png"];

また、生成された _Image オブジェクトは、不要になったらオブジェクト自身の release メソッドで破棄します。

[img_s release];
